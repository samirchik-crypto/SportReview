# FitnesReview

Многостраничный двуязычный (RU/EN) сайт с обзорами фитнес-приложений.

## Страницы
- `index.html` — главная страница и витрина.
- `apps.html` — большой каталог приложений с фильтром.
- `compare.html` — таблица сравнений.
- `guides.html` — гайды и полезные материалы.
- `about.html` — о проекте и контакты.

## Запуск
Откройте любой HTML-файл в браузере или поднимите простой сервер:

```bash
python3 -m http.server 8080
```

Затем перейдите на `http://localhost:8080`.
